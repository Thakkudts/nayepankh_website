# NayePankh Foundation Website

A modern, responsive NGO website built for **NayePankh Foundation** using React, Tailwind CSS, and Framer Motion.

## About

NayePankh Foundation is a social impact organization dedicated to creating positive change through education, community development, volunteer engagement, and awareness initiatives.

This website serves as the digital platform for showcasing the foundation's mission, programs, volunteer opportunities, certificates, donation initiatives, and contact information.

## Features

* Responsive Design
* Modern NGO-Themed UI
* Animated User Experience with Framer Motion
* About Us Section
* Programs & Initiatives
* Volunteer Registration Page
* Certificates & Registrations Page
* Donation Page
* Contact Page
* Mobile-Friendly Navigation

## Tech Stack

* React.js
* React Router DOM
* Tailwind CSS
* Framer Motion
* Vite

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd nayepankh-foundation
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
